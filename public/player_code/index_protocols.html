  • HLS

    Apple HTTP Live Streaming (HLS) specification with TS segmentation is supported for de-mux and playback. The playback supports both individual video quality as well as auto quality selection. The HLS-AES with HTTPS based simple DRM system is also provisioned. All HLS compliant VoD & Live Streaming servers are supported. 

  • Dash

    MPEG-DASH, an international standard for adaptive video streaming based on HTTP is supported for both de-mux and playback. The playback supports both individual video quality as well as auto quality selection. Both the DASH containers – fMP4 and MPEG2TS are supported for playback and Google Widevine DASH DRM system is also provisioned. All DASH compliant VoD & Live Streaming servers are supported.

  • MP4

    This format supports HTTP progressive download based playback with MP4 AV files. The codec combination of HEVC, H.264, and H.263 with AMR/AAC/AC3 valid combinations are supported.
